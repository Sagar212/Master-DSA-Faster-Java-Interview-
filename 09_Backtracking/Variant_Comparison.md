# Variant Comparison: Backtracking
## 1. Subsets — Include/skip each element. Use start index. Result at every node.
## 2. Permutations — Use every element. Use boolean[] used. Result at leaf only.
## 3. Combinations — Like subsets but with a target count. Prune early.
## 4. Constraint Satisfaction — N-Queens, Sudoku. isSafe() check before choosing.

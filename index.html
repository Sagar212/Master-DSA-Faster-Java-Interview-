Browser security policies restrict loading external scripts (like DSAPatternVisualizer.jsx) when files are opened directly via the file:// protocol.

How to Run the Visualizer

1

Use serve.bat (Recommended)

Double-click the serve.bat file we've created in this directory. It will start a local server and open the app in your browser.

2

Or Start a Server Manually

Open a terminal in this directory and run:

python -m http.server 8000

Then navigate to http://localhost:8000/index.html

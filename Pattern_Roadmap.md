# Pattern Roadmap

This roadmap provides a structured path for mastering Data Structures and Algorithms (DSA) through pattern recognition rather than rote memorization.

## 1. Fundamentals
- **01_Arrays_HashMap**: The foundation of most interview questions. Learn to trade space for time using HashMaps.
- **02_TwoPointers**: Master traversing sequences efficiently. Opposite ends, same direction, and slow/fast pointer variants.
- **03_SlidingWindow**: Essential for subarray/substring problems. Fixed and variable window variants.

## 2. Advanced Traversal
- **04_BinarySearch**: Beyond simple searching—finding boundaries, rotating arrays, and searching solution spaces.
- **05_Stack_Queue**: Monotonic stacks for next greater elements, BFS with queues.

## 3. Non-Linear Structures
- **06_Trees_Graphs**: DFS, BFS, Topological Sort, Union Find.
- **07_Heap**: Top K elements, merging K sorted lists, running median.

## 4. Complex Optimization
- **08_Core_DP**: 1D DP, 2D DP, Knapsack, Palindromes.
- **09_Backtracking**: Combinations, permutations, subsets.

## Study Workflow
1. Read the `README.md` and `Pattern_Recognition.md` for each pattern.
2. Review the `Pattern_Template.java`.
3. Solve `Easy` problems to grasp the core concept.
4. Move to `Medium` to encounter variants and edge cases.
5. Tackle `Hard` only when comfortable with `Medium` variants.

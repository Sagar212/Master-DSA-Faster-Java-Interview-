Java 8 Stream API & Collections

Highly optimized, production-grade coding solutions using lambdas and advanced Collectors.

Section 1: Numerical & Array Transformations
1. Separate Odd and Even Numbers (Partitioning) 
Map<Boolean, List<Integer>> partitioned = list.stream()
  .collect(Collectors.partitioningBy(i -> i % 2 == 0));
3. Find the Second Largest Number 
Optional<Integer> second = list.stream()
  .distinct()
  .sorted(Comparator.reverseOrder())
  .skip(1)
  .findFirst();
5. Sum of All Digits of a Number 
int sum = Stream.of(String.valueOf(num).split(""))
  .mapToInt(Integer::parseInt)
  .sum();
6. Merge Two Unsorted Arrays into a Single Sorted Sequence 
int[] merged = IntStream.concat(Arrays.stream(a), Arrays.stream(b))
  .sorted()
  .toArray();
10. Generate Fibonacci Series 
List<Integer> fib = Stream.iterate(new int[]{0, 1}, f -> new int[]{f[1], f[0] + f[1]})
  .limit(10)
  .map(f -> f[0])
  .collect(Collectors.toList());
Section 2: Text Processing & String Manipulation
11. Frequency of Each Character in a String 
Map<Character, Long> freq = str.chars()
  .mapToObj(c -> (char) c)
  .collect(Collectors.groupingBy(Function.identity(), Collectors.counting()));
12. Verify Anagram Status via Streams 
boolean isAnagram = Stream.of(s1.toLowerCase().split(""))
  .sorted().collect(Collectors.toList())
  .equals(Stream.of(s2.toLowerCase().split("")).sorted().collect(Collectors.toList()));
13. Reverse Each Word of a String 
String rev = Arrays.stream(str.split(" "))
  .map(word -> new StringBuilder(word).reverse().toString())
  .collect(Collectors.joining(" "));
Section 3: Filtering & Grouping Variations
18. Extract Duplicate Elements From a Collection 
Set<Integer> unique = new HashSet<>();
Set<Integer> duplicates = list.stream()
  .filter(i -> !unique.add(i))
  .collect(Collectors.toSet());
22. Flatten a Nested List Matrix 
List<Integer> flat = nestedList.stream()
  .flatMap(Collection::stream)
  .collect(Collectors.toList());
24. Group Employees by Department 
Map<String, List<Employee>> byDept = employeeList.stream()
  .collect(Collectors.groupingBy(Employee::getDepartment));
